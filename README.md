# Weather-App

Weather App that uses openweather.org's API to retrieve real-time data about the city of your choice. Buzz.js is used to alert users if the temperature is too cold. 

## Built With 

* HTML5
* SCSS compiled to CSS
* Vanilla JavaScript
* Open Weather API
* Buzz.js
* Love
